# bin
Utilities for Maintanance  

root/  
- archive_dir.sh - Automated .tar.zst Compression  
- download-server.py - Stable LAN file distribution server  
- extract_dir.sh - Automated .tar.zst Extraction  
- fixperms.sh - !DANGER! Fixes Permissions in a Directory (for samba)  
- flac2mp4.sh - Converts flac to video (for music tracks)  
- hash.py - Writes checksums recursively, includes .json database for easy re-hash  
- html_gen.py - Writes index.html recursively for each directory  
- rename.py - Renames each file in a directory per their hash  

! EXPERIMENTAL !  

cas/  
- cas.py - Content-Addressed Storage Engine
- cas_cli.py - Script to interact with said Storage Engine

outdated/ contains files from a home folder I had on an ancient 2010s macbook linux installation. Recovered 2026-07-14, Dated 2023-07-13  
I have not documented these files, they are grandfathered into this repository.
