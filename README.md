Utilities for Maintanance

extract_dir.sh - Automated .tar.zst Extraction  
fixperms.sh - !DANGER! Fixes Permissions in a Directory (for samba)  
flac2mp4.sh - Converts flac to video (for music tracks)  
rename.py - Renames each file in a directory per their hash
